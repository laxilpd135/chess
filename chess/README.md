# Simple Chess Game in C

## Description

This is a console-based Chess Game developed using the C programming language. The game allows users to start from either the initial chess position or an intermediate board position. Players can choose to play against another player or against the computer. The program validates moves and displays appropriate messages for invalid moves.

## Features

* Start from the initial chess position
* Start from an intermediate/custom board position
* Two-player mode
* Player vs Computer mode
* Console-based chess board display
* Turn-based gameplay
* Move validation
* Detection and notification of invalid moves
* User-friendly menu options

## Technologies Used

* C Programming Language
* GCC Compiler

## How to Run

1. Compile the program:
   gcc chess.c -o chess

2. Run the executable:
   ./chess
